# Plant-Protector
Mobile App to ease the workload while gardening. Accomplished by using ESP8266 controllers and Web data to inform user to take action in the garden.

## Installation
### For Mobil
```bash
Clone Repository
```

```bash
npm install
```
```bash
expo start
```
### For Web
```bash
Clone Repository
```

```bash
npm install
```
```bash
expo start --web
